# Srebello Cloud Calculator

This project is a simple cloud calculator web application.

## Frontend
- HTML
- CSS
- JavaScript

## Features
- Addition
- Subtraction
- Multiplication
- Division

## Project Structure

index.html – Landing page  
calculator.html – Calculator interface  
style.css – Styling  
script.js – Calculator logic  

## Future Improvements

- ASP.NET Core API backend
- AWS EC2 deployment
- NGINX reverse proxy
- HTTPS
- Domain integration
